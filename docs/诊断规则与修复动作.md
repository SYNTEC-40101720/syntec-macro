## 维护说明

- 新增诊断 code 时，请同步检查 `src/diagnosticActions.js` 是否需要 Quick Fix 或说明型 action。
- 能确定等价替换的场景使用自动 Quick Fix；需要现场语义判断的场景只提供说明型 action。
- 文案可调整，但 code 应保持稳定，避免破坏测试、去重和外部引用。
