# v2.11.0 发布规划

## 版本定位

`v2.11.0` 发布 `v2.10.0` 之后已核实的 MACRO 函数语义、保守静态诊断与回归覆盖。它不宣称覆盖控制器或模拟器运行时行为；`CALL-RUN`、Cycle 档案、图形模拟和堆栈的实机验证仍留在后续迭代。

## 发布内容

- `AXID(axis)` 使用裸轴名的补全、Hover 与说明型 warning；带引号的 `AXID("Y")` 不自动改写，也不检查实际轴配置。
- `DBLOAD/DBINSERT` 的 Cycle name 覆盖语义与 `DBDELETE` 回传码 Hover。
- 已核实的数学定义域、单位/堆栈、图形模拟函数说明及回归测试。
- 诊断规则 Markdown 方括号转义，避免间接变量形式被误解析为链接。

## 发布边界

- `GETPR`、`SETPR`、`SYSVAR` 仍没有直接 A 级函数页可确认参数范围、权限、返回值或时机条件；本版本不为这些函数新增诊断。
- 不把模拟器/控制器运行时状态升级为静态 error。
